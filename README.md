## Deploy to Heroku

Ensure that you are logged in:

```
heroku login
```

Ensure remote named heroku has been set:

```
git remote -v
```

Push changes to heroku:

```
git add .
git commit -m '...'
git push heroku main
```

Build frontend and deploy

```
npm run deploy:full
```

View logs:

```
npm run logs:prod
```
